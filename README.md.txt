Sales Performance Analysis using SQL and Python

Overview  
This project focuses on analyzing sales data to evaluate business performance and extract meaningful insights.  
It follows an end-to-end analytical workflow using SQL for querying structured data and Python for analysis and visualization.

Tools and Technologies  
• SQL (SQLite)  
• Python  
• Pandas  
• Matplotlib  
• Seaborn  

Dataset  
The dataset consists of transactional sales records containing attributes such as sales, profit, category, and time-related fields.  
It supports analysis of trends, patterns, and relationships across multiple business dimensions.

Analysis Performed  
• Monthly sales trend analysis  
• Category-wise sales and profit analysis  
• Correlation analysis between key numerical metrics  
• Visualization of performance patterns and trends  

Key Insights  
The analysis highlights variations in sales and profit across categories and time periods.  
Relationships between important metrics are explored to better understand overall business performance.

Conclusion  
This project demonstrates the use of SQL and Python to analyze structured sales data and generate clear, data-driven insights.  
It emphasizes analytical reasoning, data exploration, and effective visualization to support performance evaluation and decision-making.
